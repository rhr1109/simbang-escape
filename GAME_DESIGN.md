# 심방런 (SimbangRun) 게임 기획서

> **도망가는 남편을 잡아라!** 2D 사이드 스크롤러 러너 게임.

---

## 1. 게임 개요

- **장르**: 2D 러너 / 오락실 액션
- **플랫폼**: 웹 브라우저 (PC + 모바일 터치)
- **기술**: HTML5 Canvas + Web Audio API (순수 JS, 외부 의존 없음)
- **입력**: 키보드(PC) / 터치(모바일)
- **1회 플레이**: 12~14 스테이지 (모드 따라 다름)

### 한 줄 요약
심방(설교) 직후 도망치는 남편을 잡기 위해 아내가 달리는 추격 액션. 교회/종교 아이템을 먹어 남편을 느리게 만들어 점점 거리를 좁혀서 잡는 게임.

---

## 2. 게임 플로우

```
[메뉴 (오락실 BGM)]
      │  모드 선택 (EASY / NORMAL / HARD / CRAZY)
      ▼
[스토리 인트로]      ← ← ← ← ← ← ← ← ← ← ← ← ← ← ←
      │  Space/Enter/클릭                              │
      ▼                                                │
[스테이지 플레이 (스테이지 BGM)]                        │
      │                                                │
      ├── 남편 잡음 → [스테이지 클리어] → 다음 스테이지 ┤
      │                                                │
      ├── 남편 도망 → [게임 오버] → 같은 모드 1단계 재도전
      │
      └── 모든 스테이지 클리어 → [엔딩]
```

### 상태 전이
- `menu`: 타이틀 화면, 모드 선택
- `storyIntro`: 각 스테이지 시작 전 스토리/정보 패널
- `playing`: 실제 플레이
- `paused`: 일시정지
- `stageClear`: 스테이지 클리어 화면
- `gameOver`: 실패 화면
- `ending`: 전체 클리어 엔딩

## 3. 조작법

### 키보드
| 키 | 동작 |
|---|---|
| `↑` / `W` / `Space` | 점프 (공중에서 한 번 더 누르면 2단 점프) |
| `→` / `D` | 가속 (꾹 누르면 속도 부스트) |
| `P` / `ESC` | 플레이 중 일시정지 토글 |
| `ESC` | 일시정지/메뉴 외 화면에서 메뉴 복귀 |
| `←` `→` `↑` `↓` | 메뉴 화면에서 모드 선택 |
| `Space` / `Enter` | 확인 / 다음 진행 |

### 마우스/터치
- **메뉴**: 모드 카드 클릭/터치로 즉시 시작
- **플레이 중 아무 곳 탭**: 점프
- **우측 HUD 버튼(⏸)**: 일시정지
- **우측 HUD 버튼(🏠)**: 메뉴로 복귀
- **모바일 전용 버튼**: `JUMP`(점프) / `ACC`(가속) / `START`(상태 진행)

## 4. 난이도 모드

각 모드는 동일한 12스테이지 베이스에 배율을 곱해 난이도를 조정합니다. CRAZY만 추가 스테이지 2개가 포함된 14스테이지 구성입니다.

| 모드 | 이모지 | 남편속도 ×| 아내속도 × | 회피율 ×| 추격거리 ×| 아이템률 ×| 장애물률 ×| 스테이지 수 |
|---|---|---|---|---|---|---|---|---|
| **EASY**   | 🌱 | 1.00 | 1.00 | 1.00 | 1.00 | 1.00 | 1.00 | 12 |
| **NORMAL** | ⚡ | 1.15 | 0.98 | 1.30 | 1.08 | 0.92 | 1.15 | 12 |
| **HARD**   | 🔥 | 1.30 | 0.95 | 1.60 | 1.15 | 0.82 | 1.30 | 12 |
| **CRAZY**  | 💀 | 1.50 | 0.92 | 2.00 | 1.25 | 0.72 | 1.50 | **14** |

**배율 효과**
- `남편속도 ×`: 남편의 기본 이동 속도
- `아내속도 ×`: 아내의 기본 이동 속도 (↑ 값일수록 유리)
- `회피율 ×`: 남편이 장애물(목사/장로 등)을 점프 회피하는 확률 (상한 85%)
- `추격거리 ×`: 시작 시점의 남편과의 거리
- `아이템률 ×`: 초당 아이템 스폰 확률 (↓ 값일수록 불리)
- `장애물률 ×`: 장애물 NPC 스폰 확률

### CRAZY 예시 (스테이지 12 "옥상 추격전")
| 항목 | 기본값 | CRAZY 최종값 |
|---|---|---|
| 남편 속도 | 6.0 | **9.0** |
| 아내 속도 | 3.4 | **3.128** |
| 회피율   | 50% | **85% (상한)** |
| 추격 거리 | 780m | **975m** |

## 6. 핵심 게임 메카닉

### 6.1 추격 거리 시스템
- 각 프레임마다 `distance += (husband.currentSpeed - wifeSpeed) * 0.5` 적용
- `distance`가 **0 이하** → 남편 잡음 → 스테이지 클리어
- `distance`가 **catchDist + 400 초과** → 남편 도망 → 게임 오버
- UI 진행 바는 0(아내 승) ↔ catchDist+200(남편 승) 범위로 시각화

### 6.2 아이템 시스템
14종의 종교/교회 관련 아이템이 랜덤 위치(바닥/1단점프/2단점프 높이)에 스폰됩니다. 아내가 지나치면서 획득하면 **남편에 슬로우 효과**를 부여합니다.

#### 아이템 가뭄 방지
- 화면에 아이템이 없고 90프레임(~1.5초) 스폰이 없으면 **강제 스폰**
- 화면에 아이템이 있어도 240프레임(~4초) 스폰이 없으면 **강제 스폰**
- 덕분에 "아이템 없이 남편이 일방적으로 도망" 상황 방지

#### 아이템 목록
| 이름 | 이모지 | 효과 | 지속 | Tier | 특수 효과 타입 |
|---|---|---|---|---|---|
| 목사님의 전화 | 📞 | 슬로우 2.0 | 180프레임 | 1 | NPC 출동 (목사님) |
| 등짝 스매싱 | 👋 | 슬로우 3.5 | 120 | 1 | 등짝 스매싱 이펙트 |
| 설교 | 📖 | 슬로우 1.5 | 240 | 1 | 책 이펙트 |
| 성경 | ✝️ | 슬로우 2.0 | 200 | 1 | 빛의 심판 |
| 아내의 고함 | 📢 | 슬로우 2.5 | 150 | 2 | "여보!!!" 충격파 |
| 찬송가 | 🎵 | 슬로우 1.8 | 220 | 2 | 음표 이펙트 |
| 권사님의 잔소리 | 👵 | 슬로우 2.2 | 190 | 2 | NPC 출동 (권사님) |
| 헌금봉투 | 💰 | 슬로우 1.0 | 300 | 1 | 돈 이펙트 |
| 떡볶이 | 🍜 | 슬로우 1.5 | 180 | 2 | 음식 냄새 |
| 주일학교 아이들 | 👶 | 슬로우 2.0 | 160 | 3 | NPC 출동 (아이들) |
| 집사님의 기도 | 🙏 | 슬로우 2.8 | 140 | 3 | NPC 출동 (집사님) |
| 버스 열쇠 | 🔑 | 슬로우 3.0 | 130 | 3 | 차 열쇠 압수 |
| 안수기도 | 🤲 | 슬로우 4.0 | 100 | 4 | 안수기도 황금빛 |
| 전교인 기도회 | ⛪ | 슬로우 5.0 | 80 | 4 | 다수의 기도하는 손 강림 |

- **Tier**: 스테이지 maxTier 이하의 아이템만 등장. 뒷 스테이지일수록 강력한 아이템 해금.

#### 2단 점프 아이템
- 15%~35% 확률로 매우 높은(150~200px) 아이템이 등장하며 `2단 JUMP!` 레이블 표시. 2단 점프로만 획득 가능.

#### 아이템 이동 패턴
- `straight` (직선), `sine` (파형), `zigzag` (지그재그), `dive` (공중에서 급하강) — 후반 스테이지에서 난이도 상승.

### 6.3 장애물 NPC
| 이름 | 색상 | 칼라 |
|---|---|---|
| 목사님 | 다크네이비 | 있음 |
| 장로님 | 회청 | 없음 |
| 권사님 | 보라 | 없음 |
| 집사님 | 청록 | 없음 |
| 성도 | 녹색 | 없음 |
| 성가대원 | 진빨강 | 없음 |

- 남편이 장애물에 접근하면:
    - **회피율** 확률로 점프 회피 (성공 시 "휙!" 이펙트)
    - 아니면 장애물에 막혀 슬로우 0.8을 40프레임 받음 ("어딜!" 말풍선)
- 아내는 장애물을 **자동 통과** (흐리게 표시됨)

### 6.4 콤보 시스템
- 아이템을 연속으로 먹으면 `combo x2`, `x3` ... 최대 x5
- 슬로우 효과 배율 = `item.slowAmount * (1 + mult * 0.2)`
- 점수 = `item.slowAmount * 100 * combo`
- 3초 동안 새 아이템 안 먹으면 콤보 리셋

### 6.5 남편 감속 효과 누적
- 여러 슬로우가 동시에 쌓일 수 있음
- `husband.currentSpeed = max(0.3, baseSpeed - Σ slowEffects)`
- HUD에 각 효과와 남은 시간 표시 ("등짝 스매싱 (1.8s)")
- 남편은 감속 중 **겁먹은 상태**(scared): 땀방울, 놀란 눈, 이모지, 느린 애니메이션

## 7. UI / UX

### 7.1 메뉴 화면
- 오락실 CRT 스캔라인 + 네온 픽셀 배경
- 타이틀 "심방런" 네온 빨강 글로우 + 서브타이틀
- **모드 선택 2×2 그리드**: 각 카드에 이모지/이름/설명/12 STAGES/난이도 막대
- 방향키 네비 + 선택시 글로우 + `▶◀` 화살표
- 하단 한 줄 조작 안내

### 7.2 플레이 HUD (상단 스트립 내부)
- 좌측: `[💀CRAZY]` 모드 배지 + `12단계: 옥상 추격전` 스테이지명
- 중앙: 아내 👩 ↔ 남편 🏃 추격 진행 바 + 남은 거리(m)
- 우측(배지/이름 우측): 점수 + 총점
- 최우측: `⏸ 정지` / `🏠 나가기` 아이콘 버튼 (HUD 내부 → 플레이 영역 안 가림)
- 좌측 아래: 현재 활성 감속 효과 목록
- 우측 아래: 범례 (초록빛 = 먹기 / 흐린 사람 = 자동 통과)

### 7.3 일시정지 오버레이
- **일시정지 직전 프레임을 `getImageData`로 스냅샷**하여 `pauseSnapshot`에 저장
- 일시정지 중엔 매 프레임 `ctx.putImageData(pauseSnapshot, ...)` 로 복구 + 오버레이만 덧그림
  → 외부 상태(currentStage, items 등)가 바뀌어도 **정확히 일시정지 순간의 화면이 유지됨**
- "일시정지" 제목 + "▶ 계속하기" / "🏠 메뉴로" 버튼

### 7.4 스토리 인트로 / 클리어 / 게임오버 / 엔딩
- 전부 중앙 정렬된 패널 디자인
- 좌상단에 `← 메뉴` 복귀 버튼

### 7.5 반응형
- 캔버스 1000×600 고정, CSS `transform` 없이 width/height 스케일링
- 모바일: 하단 JUMP / ACC / START 터치 버튼 자동 노출

## 8. 오디오

### 8.1 오락실 BGM (메뉴)
- 130ms 템포의 8-비트 스타일
- **3 레이어**: square(멜로디) + triangle(아르페지오) + square(베이스 ×0.5)
- C4 기준, 상승/하강 32노트 루프

### 8.2 스테이지 BGM (14종)
- 각 스테이지마다 고유 템포/파형/음계
- 2 레이어: 메인 파형(멜로디) + sine(베이스 ×0.5)
- 음표 위에서 주파수 자동 변경, 80% sustain + 70% fade 엔벨로프
- Web Audio API로 실시간 생성 (외부 오디오 파일 불필요)
- CRAZY 전용 2곡은 **MP3 추출본**도 함께 제공 (오프라인 참고용)
    - `assets/bgm/bgm-stage12-rooftop.mp3` — 130bpm sawtooth, 어둡게 상승하는 불안감
    - `assets/bgm/bgm-stage13-subway.mp3` — 120bpm square, 터널 울림 베이스
- 재생 스크립트: `tools/render-bgm.mjs` (오프라인 렌더링 → `@breezystack/lamejs`로 MP3 인코딩)

### 8.3 효과음
- 별도 SFX는 없음. 충격 시 화면 셰이크 + 파티클/텍스트 이펙트로 체감 대체.

## 9. 점수와 등급

### 스테이지 점수
- 아이템 획득마다 `slowAmount * 100 * combo` 점수 누적
- 콤보 유지가 높은 점수의 핵심
